# Tutorials

Worked examples showing how to go from a raw single-cell dataset to a trained
scLEMBAS model and downstream analysis.

## Available tutorials

- **[Kang et al. example](Kang_example.ipynb)** — walkthrough on the
  Kang et al. (2017) IFN-β stimulation dataset: model training and evaluation of
  predicted responses.

The source notebooks live in
`docs/tutorials/` and can be downloaded and run locally after
[installing scLEMBAS](../installation.md).
